## Install

```sh
yarn add --dev eslint typescript babel-plugin-module-resolver @babel/core @ymkz/eslint-config
```

## Usage

Write the following code in `.eslintrc.json`

```.eslintrc.yml
extends:
  - '@ymkz/eslint-config'
```

## Requirements

- `eslint` >= `6.8.0`
- `typescript` >= `3.7.4`
