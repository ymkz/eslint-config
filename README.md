## Install

```sh
yarn add --dev --exact eslint typescript @ymkz/eslint-config
```

## Usage

Write the following code in `.eslintrc.json`

```
{
  "extends": ["@ymkz/eslint-config"]
}
```

## Requirements

- `eslint` >= `6.8.0`
- `typescript` >= `3.7.4`
