## Install

```sh
yarn add --dev @ymkz/eslint-config
```

## Usage

Write the following code in `.eslintrc.yml`

```.eslintrc.yml
root: true
extends:
  - '@ymkz/eslint-config'
  - '@ymkz/eslint-config/react'
  - '@ymkz/eslint-config/import'
  - '@ymkz/eslint-config/typescript'
```

## Requirements

- `eslint` >= `6.8.0`
- `prettier` >= `1.19.0`
